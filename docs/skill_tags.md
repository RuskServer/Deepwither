# スキルタグシステム仕様書

本ドキュメントでは、Deepwitherのスキルに設定可能な「タグシステム」に関する仕様を定義します。  
タグはスキルツリーGUIやスキル割り当てGUIといったユーザー接点において、そのスキルの特性や運用方法を一目で初心者へ伝える役割を果たします。

## 概要

スキルごとのメタデータとして4つのカテゴリ（役割、戦術、特性、制限）に分けたタグを持たせることができます。
設定されたタグは、GUI上でスキルの基本情報の直上にコンパクト（最大2行）にまとめられ、カテゴリーごとに色分けして表示されます。

## YAML設定の記述方法

スキルのYAML設定ファイル（例: `skills/flash_slash.yml`）に、`tags` セクションを追加することで設定します。
大文字小文字は区別されず、ケバブケース（ハイフン区切り）で記述します。各カテゴリに複数のタグを指定可能です。

```yaml
name: "メテオ"
tags:
  role: 
    - "ATTACK"
  tactics: 
    - "BURST"
  scaling: 
    - "MAGICAL"
  constraint: 
    - "HIGH-COST"
    - "LONG-CD"
```

## カテゴリとタグ一覧

### 1. 役割 (Role)
スキルの基本的な立ち位置や枠割を示します。

| タグ名 | 表示名 | 色 | 説明 |
| :--- | :--- | :--- | :--- |
| `ATTACK` | 攻撃 | 赤 (`RED`) | 純粋なダメージソースとして機能するスキル（例: flash_slash） |
| `CONTROL` | 妨害 | 黄 (`YELLOW`) | 敵の行動や移動を制限するデバフスキル（例: stunbolt, binding_slash） |
| `DEFENSE` | 防御 | 青 (`BLUE`) | 自身の生存率を高める防御・シールドスキル（例: abyss_shield, frost_armor） |
| `SUPPORT` | 支援 | 緑 (`GREEN`) | 味方の回復やバフを付与するサポートスキル（例: celestial_heal, war_roar） |
| `UTILITY` | 機能 | 紫 (`LIGHT_PURPLE`) | 移動や特殊効果など、立ち回りを強化するスキル（例: iron_hook, charge_warrior） |

### 2. 戦術 (Tactics)
PvPや特化ビルドにおける「どの相手、どの状況に刺さるか」という戦術的要素を示します。

| タグ名 | 表示名 | 色 | 説明 |
| :--- | :--- | :--- | :--- |
| `BURST` | 瞬発 | 金 (`GOLD`) | 一撃の威力が極めて高く、瞬間火力を期待できる（例: meteor） |
| `DISPLACE` | 位置操作 | 深海 (`DARK_AQUA`) | 敵を動かす（ノックバック・引き寄せ）効果を持つ（例: iron_hook） |
| `ANTI-TANK` | 対重装 | 濃赤 (`DARK_RED`) | 防御力を無視する、または割合ダメージを与える効果を持つ |
| `MOBILITY` | 機動力 | 水色 (`AQUA`) | 移動速度上昇やワープなど、足回りを強化する（例: quick_reaction_reflex） |
| `DISPEL` | 解除 | 白 (`WHITE`) | 敵のバフを解除する、または味方のデバフを消散させる |

### 3. 特性 (Scaling)
武器やアーティファクトとのシナジー（ビルドの方向性）を示します。

| タグ名 | 表示名 | 色 | 説明 |
| :--- | :--- | :--- | :--- |
| `PHYSICAL` | 物理 | 金 (`GOLD`) | 攻撃力（ATK）ステータスに大きく依存する |
| `MAGICAL` | 魔法 | 明紫 (`LIGHT_PURPLE`) | 魔力（Magic Damage）ステータスに大きく依存する |
| `HYBRID` | 複合 | 黄 (`YELLOW`) | 攻撃力・魔力の両方のステータスが影響する |
| `CDR-HEAVY` | 高回転 | 水色 (`AQUA`) | 元のCDが短く、クールダウン短縮ステータスの恩恵が大きい |

### 4. 制限 (Constraint)
強力なスキルが持つ「リスク・デメリット」を可視化し、スロット構成の重みを調整する指標です。

| タグ名 | 表示名 | 色 | 説明 |
| :--- | :--- | :--- | :--- |
| `CHANNELING` | 詠唱 | 濃紫 (`DARK_PURPLE`) | 発動中に隙ができる（移動速度低下や詠唱中の中断リスク） |
| `HIGH-COST` | 高燃費 | 濃赤 (`DARK_RED`) | マナ消費が激しく、連発が難しい（マナ管理が必須） |
| `LONG-CD` | 長大CD | 灰 (`GRAY`) | 一度使うとしばらく使えない「切り札」的なスキル |

## 表示仕様 (GUI Lore)

タグが設定されている場合、スキルアイテム等のLoreの先頭（スキル名・説明文の上）に以下のフォーマットで表示されます。

```
役割: [攻撃] [妨害]  戦術: [瞬発]
特性: [魔法]  制限: [高燃費] [詠唱]

スキル説明文1行目...
スキル説明文2行目...
```
※複数指定されている場合は横に並んで表示され、未指定のカテゴリについては項目自体が非表示になります。
